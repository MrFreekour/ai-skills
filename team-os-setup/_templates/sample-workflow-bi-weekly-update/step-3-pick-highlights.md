# Step 3 — Pick highlights

Take the draft from Step 2 and prune. The audience reads in 5 minutes — anything that doesn't earn its place gets cut.

## Pruning rules

1. **Headline must answer:** "What's the one thing leadership should know?" If it doesn't, rewrite it.
2. **Shipped section: max 5 bullets.** Group related items if needed. If you have more than 5, you're listing minor things — cut them.
3. **Decisions: max 3.** A decision worth surfacing has stakes — if leadership wouldn't ask "why did we do that?", it doesn't belong here.
4. **Blockers / asks: max 3.** If everything is on fire, nothing is. Pick the actual asks.
5. **Customer signal: max 1 pattern.** If you have multiple, pick the most strategic.

## Sanity check

Read the draft aloud (or have Claude read it back). Test:
- Does the headline give the audience what they need in one sentence?
- Could you delete any bullet without changing what the audience does next? If yes, delete it.
- Is anything a status update without a "so what"? Cut it.

## Output of this step

The pruned draft, ready for formatting in Step 4.
